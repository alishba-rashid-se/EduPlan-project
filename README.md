# 🐞 Defects Identified

The following defects were discovered during manual testing:

1. Login accepted incorrect passwords.
2. Login accepted invalid or unregistered email addresses.
3. Previous user session remained active after switching accounts.
4. Demo Login displayed incorrect user profile information.
5. Dashboard did not display updated task information after re-login.
6. Upcoming Deadlines section displayed only three tasks even when additional upcoming tasks existed.

All defects were documented in Jira, fixed, verified, and successfully passed during regression testing.
